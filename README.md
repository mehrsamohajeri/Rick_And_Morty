# Rick and Morty - Character Browser

Browse characters from the Rick and Morty series using the [Rick and Morty API](https://rickandmortyapi.com/documentation).

## Features

- Character list with search and pagination
- Character detail page (episodes, location)
- SSR, responsive design, image loading states
